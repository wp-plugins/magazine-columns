Magazine Columns

Version 1.0.1
WordPress plugin

== Installation ==

1. Unzip the magazine-columns.zip file.
2. Upload the magazine-columns folder to the /wp-content/plugins/ directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Use the columns quicktag in the HTML editor to add the <!--columns--> tag to your posts or pages.
